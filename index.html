Proyecto 2DAM

Aprende música sin barreras

Wavii es la plataforma integral (SaaS Freemium) que conecta a estudiantes con profesores certificados y fomenta una comunidad activa de aprendizaje musical.

Descubre Más Leer Memoria

¿Por qué elegir Wavii?

Nuestra plataforma resuelve la falta de recursos accesibles con tecnología de vanguardia.

Profesores Certificados

Encuentra tutores homologados usando el GPS de tu dispositivo para clases presenciales o remotas.

Comunidad Activa

Interactúa con otros estudiantes, forma bandas y comparte tus grabaciones para fomentar el aprendizaje conjunto.

Modo Offline

Revisa tus lecciones y edita tu perfil sin conexión. Tus datos se sincronizan automáticamente.

Arquitectura del Sistema

Una solución robusta y escalable que integra Frontend, Backend y ERP.

Mobile App

React Native & Expo SDK 54
Aplicación multiplataforma con Zustand para estado, navegación reactiva y temas dinámicos.

Backend API

Spring Boot 3 & Java 21
Servicios REST securizados con JWT, integración con PostgreSQL y MongoDB, y pasarela de pagos Stripe.

ERP & CRM

Odoo v17
Gestión interna, contabilidad y validación de profesores certificados mediante webhooks bidireccionales.
